ExpenseVerse – Smart Expense Tracker
ExpenseVerse is a responsive and interactive web application that helps users manage and visualize their daily, monthly, and yearly expenses.
It allows users to add, edit, delete, and track expenses with category-wise breakdowns, sort and filter features, and real-time visualization using Chart.js.
This project showcases CRUD operations, localStorage persistence, dark mode UI, and modern responsive design built using HTML, CSS, Bootstrap, and JavaScript.

Overview : 
ExpenseVerse is a modern expense tracking web app designed to help users monitor their financial activities easily. 
It allows users to add detailed expense records, view them in different time-based views (daily, monthly, yearly), and visualize their spending distribution through colorful pie charts.
The project demonstrates key front-end development concepts such as DOM manipulation, state management, localStorage integration, and responsive UI design.

Features :
>> Add, Edit & Delete Expenses – Perform full CRUD operations for managing your expenses.
>> Daily / Monthly / Yearly Views – Switch between multiple views to analyze spending patterns.
>> Real-Time Charts – Pie chart visualizations powered by Chart.js.
>> Dark Mode – Toggle between light and dark themes instantly.
>> Sorting Options – Sort expenses by amount or date in ascending or descending order.
>> LocalStorage Persistence – Data remains saved even after reloading the page.
>> Responsive UI – Works seamlessly across mobile, tablet, and desktop devices.

Tech Stack : 
Frontend: HTML5, CSS3, JavaScript 
Framework: Bootstrap 5
Charts: Chart.js
Icons: Font Awesome
Storage: Browser LocalStorage

Live demo : https://expense-verse.vercel.app/

Usage
1.Add your expenses by filling in the description, amount, category, and date fields.
2.View expenses by Daily, Monthly, or Yearly tabs.
3.Sort expenses by Amount or Date.
4.Switch between Light Mode and Dark Mode for a better visual experience.
5.Observe real-time expense distribution through the pie chart at the bottom of the page.
